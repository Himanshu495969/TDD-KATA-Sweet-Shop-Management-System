This project implements a Sweet Shop Management System following Test-Driven Development (TDD) principles.
It allows you to manage sweets inventory, user authentication, and admin functionalities efficiently.

Key features include:

Adding, updating, and deleting sweets

Searching and sorting sweets

Inventory management (stock, restock)

User authentication and authorization

Dashboard and admin panel

The system is implemented using Node.js, Express, MongoDB for backend and React with TypeScript for frontend.

ðŸš€ Getting Started
1. Clone the repository
git clone https://github.com/Himanshu495969/TDD-KATA-Sweet-Shop-Management-System.git
cd TDD-KATA-Sweet-Shop-Management-System

2. Setup Backend
cd backend
npm install


Create a .env file based on .env.example and set your MongoDB connection string and JWT secret.

Start the server:

npm start

3. Setup Frontend
cd frontend
npm install
npm start


Open http://localhost:3000
 to view the app in the browser.
