# TDD-KATA Sweet Shop Management System

## ðŸ“– Project Overview
This project implements a **Sweet Shop Management System** following **Test-Driven Development (TDD)** principles.  
It allows you to manage sweets inventory, user authentication, and admin functionalities efficiently.  

Key features include:
- Adding, updating, and deleting sweets  
- Searching and sorting sweets  
- Inventory management (stock, restock)  
- User authentication and authorization  
- Dashboard and admin panel  

The system is implemented using **Node.js, Express, MongoDB** for backend and **React with TypeScript** for frontend.
